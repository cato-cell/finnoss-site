Tannlegene Heggedalsenteret

Tannlege i Heggedal med fokus på trygg behandling og god oppfølging — på dine premisser.

Timebestilling kan gjøres via telefon, SMS eller skjema.

Hva er Tannlegene Heggedalsenteret?

Dette er et lokalt tannlegekontor i Heggedal med mål om å gjøre tannlegebesøk så forutsigbart og trygt som mulig. Du kan ta kontakt på den måten som passer deg best — ring, SMS eller e-post.

Åpningstider

  • Mandag og onsdag–fredag: 08:00–16:00
  • Tirsdag: 08:00–20:00

Ved behov kan du spørre om tider utenom åpningstidene.

Spørsmål om Tannlegene Heggedalsenteret

Hvor ligger Tannlegene Heggedalsenteret?
Tannlegene Heggedalsenteret ligger på Heggedalsveien 301 i Heggedal sentrum i Asker kommune, kort vei fra Heggedal stasjon.
Når er Tannlegene Heggedalsenteret åpent?
Tannlegene Heggedalsenteret er åpent mandag og onsdag–fredag 08:00–16:00, tirsdag 08:00–20:00. Lørdag–søndag er stengt.
Hva tilbyr Tannlegene Heggedalsenteret?
Tannlege i Heggedal med fokus på trygg behandling og god oppfølging — på dine premisser.
Hvordan bestiller jeg time hos Tannlegene Heggedalsenteret?
Ring 915 76 940 eller bestill time via nettsiden deres.
