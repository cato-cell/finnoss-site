Heggedal Legesenter

Fastlegetjenester i Heggedal. Bestill time via Helsenorge, eller bruk telefontime når det passer bedre.

Trevaren 3 • 1389 Heggedal

Dette kan du bruke legesenteret til

Heggedal Legesenter tilbyr fastlegetjenester og oppfølging i hverdagen – fra vanlige konsultasjoner til vurderinger, attester og praktiske helsetjenester.

Bestill time og kontakt

Du kan bestille time via Helsenorge. For problemstillinger som kan løses uten oppmøte, kan du bestille vanlig time og merke i notatfeltet at du ønsker telefonkonsultasjon (legg inn telefonnummer), så ringer legen deg.

Merk: Legesenteret opplyser at e-konsultasjon i perioder kan være deaktivert. Da brukes vanlig time og telefontime.

Praktisk informasjon

Åpningstider: Mandag–torsdag 08:30–15:15, fredag 08:30–15:00. Lørdag/søndag stengt.

Ved akutt sykdom: ring legevakt 116 117, og ved akutt nød 113.

Digitale tjenester

Via Helsenorge kan du typisk bestille time og fornye resept. Legesenteret informerer også om at de bruker KI for talegjenkjenning og transkribering for å redusere administrasjon og frigjøre mer tid til pasientene.
